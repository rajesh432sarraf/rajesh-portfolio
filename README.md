# RajeshOS | Developer Operating System

A premium, high-performance developer portfolio built with **Next.js 15**, **Tailwind CSS v4**, and **Framer Motion**.

## 🚀 Concept: DevOS
This portfolio is designed as an intelligent software interface (Operating System) that represents the developer’s thinking process, problem-solving ability, and engineering mindset. It avoids traditional resume layouts in favor of a modular, "dashboard" approach.

### Key Features
- **3D Cyber Grid Horizon:** A high-impact 3D-perspective background with scanning pulse effects.
- **System Initialization:** A terminal-style boot sequence that initializes kernel modules.
- **Modular Dashboard:** A window-managed interface allowing users to access Projects, Skills, and Identity pods.
- **Verified Identity Kernel:** A dedicated profile interface with unique animated frame effects.
- **Code Intelligence:** Real-time (simulated) memory and activity metrics tracking.
- **AI Interaction:** A simulated terminal chatbot for user engagement.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router, Turbopack)
- **Styling:** Tailwind CSS v4 (Pure Vanilla CSS)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Canvas Logic:** HTML5 Canvas for high-performance background motion.

## 🎨 Design System
- **Background:** `#1C1C1C` (Rich Graphite)
- **Accent:** `#C6A15B` (Muted Gold)
- **Secondary:** `#E6D3A3` (Warm Sand)
- **Typography:** Inter, Outfit (Headings), JetBrains Mono (Technical text).

## 📦 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. View at `http://localhost:3000`
